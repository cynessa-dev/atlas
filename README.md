## The Engine

In every game, there's engine. This one will use **Godot 4**. I've used it and mastered, so the development process will be much more quicker. There's a framework called Phaser.js or just Phaser, it's basically a Game Engine built for the Web. So, why not use it? I did used it and tried it out, but I noticed some vulnerabilities. Apparently, it was kind of outdated and the packagess weren't looking too good. It's a cool framework, I wish I could have given it a go, but there's just too much critical and severe vulnerabilities. Therefore, I opted with Godot 4 using Compatibility as my Renderer.
